# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Be excellent to each other. Disagreement is fine; harassment, slurs, and
personal attacks are not. Project maintainers may remove, edit, or reject
contributions and may ban contributors for behavior they deem inappropriate.

To report a concern, contact the project maintainers privately (see
`SECURITY.md` for the contact path until a dedicated address exists).
